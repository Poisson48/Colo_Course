# Colo_Course

Liste de courses partagée pour couples et colocataires — sans compte, sans abonnement,
sans serveur à payer.

- Appairage par QR code, chiffrement de bout en bout.
- Chacun garde une copie locale complète ; synchronisation automatique à chaque modification.
- Fonctionne hors ligne, gère les modifications simultanées (CRDT).
- Plusieurs listes, plusieurs participants par liste.
- Qt 6 / QML + C++. Cible Android (et desktop Linux).

**État : en développement.** Voir [docs/PLAN.md](docs/PLAN.md) pour le plan et le workflow.
